## Getting Started

[**Airdrop**](https://airdrop.eztools.xyz/) is an easy-to-use giveaway tool built on the Harmony chain. 
<br />
Users can now bulk send tokens in giveaways. Currently HRC20, HRC721 and HRC1155 are supported.
